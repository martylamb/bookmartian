Bookmartian Help

Basic Search

The simplest way to search the bookmartian is through tags. By default, when you type a word (or multiple words) in the search box and hit <ENTER> bookmartian will search for bookmarks that have been tagged with all of the tags you typed.

For example, searching for "code" will return all of the bookmarks tagged with the word "code". Searching for "code javascript" will return all bookmarks tagged with both "code" AND "javascript".

Advanced Search

More advanced queries can be constructed by adding one or more of the following query terms to the search.

example 1
visited-since:1w as:recent-visits by:most-recently-visited
returns a query named "recent-visits" with bookmarks visited within the last week, sorted with the most recently visited at the top.

example 2
code visited-since:1w by:most-visited
returns bookmarks tagged with "code" that were visited within the last week, sorted with the most frequently visited at the top.

example 3
by:most-recently-created limit:10
returns the last ten bookmarks created.

example 4
by:most-recently-created limit:10 programming
of the last ten bookmarks created, the ones tagged "programming".

queryterm:modifier description
[tagged:]TAG Default basic search term

Results will be limited to only those bookmarks tagged as TAG. The query term "tagged:" is optional.

TAG is any word or space-separated list of words.

as:NAME Query name.

Result set will be returned with the provided NAME.

by:SORT-EXPR Sort modifier.

Results will be sorted as indicated by the SORT-EXPR.

SORT-EXPR

least-recently-createdsave date, descending
least-recently-modifiededit date, descending
least-recently-visitedvisit date, descending
most-recently-createdsave date, ascending
most-recently-modifiededit date, ascending
most-recently-visitedvisit date, ascending
least-visitedvisit count, ascending
most-visitedvisit count, descending
titlealphabetically by title, ascending (case-insensitive)
urlalphabetically by url, ascending (case-insensitive)
created-before:DATE-EXPR Filter on date created.

Results will be limited to only those bookmarks created before the specified expression. The expression may be an explicit date or a relative date such as "1y3m" meaning "1 year 3 months ago".

DATE-EXPR

yyyy/MM/ddan explicit date
todaylimit to today
yesterdaylimit to yesterday
#y# years ago
#m# months ago
#w# weeks ago
#d# days ago
created-since:DATE-EXPR Filter on date created.

Use the "created-after" query term and a DATE-EXPR to only return those bookmarks added since a specified date.

created[-on]:DATE-EXPR Filter on date created.

Use the "created" query term and a DATE-EXPR to only return those bookmarks added on a specified date.

is:META-EXPR Filter on specific metadata attributes of your bookmarks.

Use the "is" query term and a META-EXPR to only return those bookmarks that meet certain other conditions.

META-EXPR

securereturn only https/secure sites.
taggedreturn only bookmarks with assigned tags.
untaggedreturn only bookmarks without assigned tags.
limit:N Restrict the number of results returned.

Use the "limit" query term and a number, N, to restrict the total number of results returned by the query.

modified-before:DATE-EXPR Filter on date edited.

Use the "modified-before" query term and a DATE-EXPR to only return those bookmarks edited before a specified date.

modified-since:DATE-EXPR Filter on date edited.

Use the "modified-since" query term and a DATE-EXPR to only return those bookmarks edited since a specified date.

modified[-on]:DATE-EXPR Filter on date edited.

Use the "modified" query term and a DATE-EXPR to only return those bookmarks edited on a specified date.

site:SITE Search by hostname

Results will be limited to only those bookmarks with urls hosted at foo.com or its subdomains (e.g. searching for "foo.com" will match bookmarks for "foo.com", "www.foo.com", and "www.bar.foo.com").

visited-before:DATE-EXPR Filter on date visited.

Use the "visited-before" query term and a DATE-EXPR to only return those bookmarks visited before a specified date.

visited-since:DATE-EXPR Filter on date visited.

Use the "visited-since" query term and a DATE-EXPR to only return those bookmarks visited since a specified date.

visited[-on]:DATE-EXPR Filter on date visited.

Use the "visited" query term and a DATE-EXPR to only return those bookmarks visited on a specified date.

visits-over:N Filter on number of time visited.

Use the "visits-over" query term and a number, N, to only return those bookmarks visited more than the specified number of times.

visits-under:N Filter on number of time visited.

Use the "visits-under" query term and a number, N, to only return those bookmarks visited fewer than the specified number of times.

visits:N Filter on number of time visited.

Use the "visits" query term and a number, N, to only return those bookmarks visited exactly the specified number of times.
