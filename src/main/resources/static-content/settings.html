The bookmartian is a self-hosted, personal bookmark database intended to make saving, finding, and using bookmarks simple but powerful. Designed to be easily hosted on your home machine or on a cloud server, bookmartian is self-contained with no external software dependencies.

Enjoy!

You can import bookmarks into Bookmartian from an existing file. The file must be formatted in the Netscape Bookmark file format.

Step 1. Find the file on your local file system.

Step 2. Provide one or more tags that should automatically applied to the bookmarks in this import. Separate the tags with spaces or commas. Tags prefixed with a period (.) will be created, but will not be visible in your tag cloud.

Step 3. Upload the file for import!
